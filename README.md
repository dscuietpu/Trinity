# 🚀 RaiseIt — Campus Issue Reporting Platform

RaiseIt is a community-driven platform where students can report campus issues, validate them collectively, and drive action through visibility and engagement.

---

## 🎯 Problem Statement

Campus issues like:
- Poor WiFi
- Dirty canteens
- Parking problems

are often:
- Scattered across conversations
- Not tracked
- Ignored by authorities

RaiseIt solves this by making issues **public, trackable, and actionable**.

---

## ✨ Features

### 📝 Issue Reporting
- Report issues with title, description, category, and building
- Upload supporting evidence (images/videos)
- Option to post anonymously

### 🤝 Community Validation
- Upvote issues to highlight importance
- Comment system for discussion

### 🧠 AI Moderation System
- Detects and blocks:
  - Spam
  - Abuse / harassment
  - Hate speech
  - Irrelevant content
- Uses **Gemini AI + rule-based filtering**
- Fail-safe handling for smooth UX

### 🔐 Secure Actions
- Only the original reporter can delete their issue
- Backend validation for authorization

### ⚡ Modern UI
- Clean, responsive frontend built with modern tooling
- Smooth user experience
